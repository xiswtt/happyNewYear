# Happy New Year!

простая, яркая и интерактивная веб-страница для празднования Нового года. создана на чистом коде HTML CSS JS, чтобы подарить праздничное настроение.

[Посмотреть живую страницу](https://xiswtt.github.io/happyNewYear/)

## Особенности

*   **Праздничный дизайн**: елочки в снежке, летающий дед мороз, фейерверки и главная праздничная ель
*   **Интерактивность**: возможность украсить елку - гирляндой и звездой
